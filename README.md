## 注意事项
可在C/C++源文件目录中使用下方指令编译且运行程序:
- g++ fileName.cpp -o fileName && fileName


若在编译过程中出现中文乱码现象, 可以如下操作:
- 使用编辑器将代码文件编码格式以 `GBK` 格式保存
- 在编译时添加参数 `-fexec-charset=GBK` 以 `GBK` 格式编译C/C++源文件 (推荐)
